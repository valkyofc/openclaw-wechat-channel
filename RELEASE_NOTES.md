# Release Notes

## v1.0.0 (2026-03-11)

### 🎉 初始版本发布

OpenClaw WeChat Channel 首个正式发布版本！

### ✨ 功能特性

#### 消息支持
- ✅ 文本消息自动回复
- ✅ 私聊智能对话
- ✅ 群聊支持（可配置 @提及回复或全部回复）
- ✅ 图片消息识别与理解
- ✅ 文件消息处理（支持 txt/csv/md/json 等文本文件）
- ✅ 语音消息支持
- ✅ 引用回复功能

#### 智能功能
- ✅ 自消息过滤，避免 AI 回复自己的消息
- ✅ 群聊白名单/黑名单机制
- ✅ 多账号会话隔离
- ✅ AI 多媒体内容理解
- ✅ 文件内容阅读与总结

#### 系统功能
- ✅ 完整的 RESTful API
- ✅ WebSocket 实时消息推送
- ✅ 可视化配置管理工具
- ✅ 实时日志监控
- ✅ 自动重连机制
- ✅ 服务健康检查

### 🚀 快速开始

详见 [README.md](README.md) 中的快速开始教程，零基础 10 分钟完成部署。

### 📦 系统要求

- Windows 10/11
- Python 3.11+
- 微信 PC 版
- OpenClaw Gateway

### 🔧 安装方式

```bash
git clone https://github.com/SEUWanglibo/openclaw-wechat-channel.git
cd openclaw-wechat-channel
scripts\install.bat
```
